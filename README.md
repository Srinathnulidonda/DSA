A comprehensive, interactive web application for mastering Data Structures and Algorithms with a structured 14-week roadmap.

## 🌟 Features

### Core Features
- **📚 14-Week Structured Roadmap** - Complete DSA curriculum from basics to advanced
- **✅ Progress Tracking** - Track daily progress with checkboxes and visual indicators
- **📅 Calendar View** - Visualize your learning schedule
- **📝 Cloud Notes** - Take and sync notes with Firebase
- **🎯 Projects Portfolio** - 14 hands-on projects
- **⏱️ Pomodoro Timer** - Built-in study timer
- **🌙 Dark Mode** - Eye-friendly dark theme
- **📱 Mobile Responsive** - Works on all devices

### Advanced Features
- **☁️ Cloud Sync** - Firebase integration for data persistence
- **📊 Progress Analytics** - Charts and statistics
- **🏆 Achievements** - Gamification elements
- **📤 Export/Import** - Backup your progress
- **🔍 Search** - Find topics and resources quickly
- **⚡ Offline Support** - Works without internet
